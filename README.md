
# AI Competitor Parser Project

## Структура проекта
1.  **Лендинг**: React + Tailwind (в корне проекта).
2.  **Telegram-бот**: Python + aiogram (папка `/bot`).

## Быстрый старт - Лендинг
Для запуска лендинга (веб-версия):
1.  Убедитесь, что у вас установлен Node.js.
2.  Запустите сервер разработки.

## Быстрый старт - Бот
Для запуска бота:
1.  Установите Python 3.10+.
2.  Установите зависимости: `pip install aiogram sqlalchemy alembic`.
3.  Создайте файл `.env` в папке `/bot` и добавьте:
    ```
    TELEGRAM_BOT_TOKEN=ваш_токен
    DATABASE_URL=sqlite:///./app.db
    PAYMENT_PROVIDER_TOKEN=ваш_токен_платежки
    ```
4.  Запустите бота: `python bot/main.py`.

## Настройка платежей
Используется **Telegram Payments**.
1. Получите токен провайдера у @BotFather (например, ЮKassa или Сбер).
2. Пропишите его в конфиг бота.

## Приватность
Сервис собирает минимум данных:
- TG ID / Username
- Имя
- Нишу (для релевантности контента)

## Ограничения сервиса (Safety & Scope)
- Не является медицинской консультацией.
- Не гарантирует 100% точность (AI-генерация требует проверки).
- Не заменяет профессиональную психотерапию.
- Не выполняет противозаконных действий (взлом и др.).
